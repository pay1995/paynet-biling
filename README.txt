PAY.Net Biling - RT/RW Net Billing App (HTML Version)

CATATAN:
1. Simpan semua file ini di satu folder.
2. Tambahkan file gambar QR DANA Anda dengan nama: qr_dana.png
   (ambil dari screenshot QR DANA dan simpan sebagai png).
3. Buka index.html dengan browser (Chrome/Edge) untuk mulai memakai aplikasi.
4. Data tersimpan di localStorage browser (hanya di perangkat ini).

Halaman:
- index.html       : Dashboard
- pelanggan.html   : Input & daftar pelanggan
- tagihan.html     : Input tagihan, kirim WA, cetak struk 58mm
- history.html     : History + export PDF
- pendapatan.html  : Total pendapatan (status LUNAS saja)
- backup.html      : Backup & restore data (JSON)
